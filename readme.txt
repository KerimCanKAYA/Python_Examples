Bu programın amacı python üzerinde, girilen şifreyi kontrol edip email ile aynı satır (yani birbakıma id'de denebilir.)numarasında 2 farklı dosyada kayıt etmeye yaramaktadır.
İstenilirse e-mail ve şifre aynı dosya ve satıe üzerindede kayıt edilebilir.



The purpose of this program is to check the entered password on python and save it in 2 different files on the same line as the email (ie it can be called id in a way).
If desired, e-mail and password can also be saved on the same file and website.
